# 发布到 GitHub Pages

1. 在 GitHub 创建一个公开仓库，例如 `cook-pwa`。
2. 将本文件夹中的所有文件上传到仓库根目录。
3. 打开仓库的 `Settings → Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，文件夹选择 `/ (root)`，然后保存。
6. 等待部署完成，地址通常为 `https://你的用户名.github.io/cook-pwa/`。

首次打开后，用两个 Supabase 用户之一登录。Android 可从浏览器菜单安装；iPhone 使用 Safari 的“添加到主屏幕”。
